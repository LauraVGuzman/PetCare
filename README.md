# PetCare - Gestor de Mascotas 🐾

**Desarrolladores:** Andres Felipe Bolívar Osorio y Laura Valentina Guzmán Morales

## Descripción del Proyecto
PetCare es una aplicación móvil desarrollada con React Native CLI para la gestión integral de mascotas. Permite registrar nuevos animales mediante un formulario validado, visualizar el listado completo, consultar detalles específicos con paso de parámetros y acceder a una sección de consejos de cuidado que se actualizan automáticamente de forma cíclica.

## Tecnologías Utilizadas
* **Framework:** React Native CLI
* **Lenguaje:** JavaScript
* **Navegación:** React Navigation (Stack Navigation & Bottom Tabs Navigator)
* **Gestión de Estado:** React Hooks (`useState`, `useEffect`)
* **Estilos:** `StyleSheet` de React Native
